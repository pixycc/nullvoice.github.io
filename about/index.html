// site

this site is built wtih the Hugo framework. it is based on no existing template, rather one i built entirely myself. you can view the source here, though i highly recommend against it. it is an utter fucking mess. eventually i might clean it up and make it a public template tho.

i made and rendered the background myself in Blender, then crunched it down to 16 colours (shades of grey). i considered transcoding to more efficient codecs (lossless webp, jpeg-xl) but the size improvement was so miniscule i dropped them.
